# Bigquery-ecommerce-analytics
SQL analysis project using Google BigQuery on Google Analytics e-commerce dataset, including traffic, revenue, conversion rate, and cohort analysis.
# BigQuery E-commerce Analytics Project

This project contains SQL queries built in Google BigQuery using the Google Analytics sample dataset.

## Objective
Analyze e-commerce performance using SQL by exploring:
- Website traffic (visits, pageviews)
- Transaction and conversion rates
- Revenue by traffic source and device
- Customer purchasing behavior
- Cohort analysis (view → add to cart → purchase)

## Dataset
- Source: Google Analytics Sample Dataset
- Table: bigquery-public-data.google_analytics_sample.ga_sessions_*

## Key Skills Used
- BigQuery SQL
- UNNEST (handling nested and repeated fields)
- Aggregations and GROUP BY
- Window functions (cumulative revenue)
- Joins and filtering
- Cohort and funnel analysis

## Queries Included
1. Monthly visits, pageviews, transactions
2. Bounce rate by traffic source
3. Revenue by week and month
4. Conversion rate analysis
5. Pageviews by purchaser type
6. Transactions per user
7. Revenue by device
8. Product purchase analysis
9. Funnel (cohort) analysis
10. Weekly and cumulative revenue

## Tools
- Google BigQuery
